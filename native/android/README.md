# Klip4ge — Android Native Shell

This directory holds the Android native project generated by Capacitor.

## Setup (run once)

```bash
# From project root
npm install
npm run build

# Add Android platform (generates this project)
npx cap add android

# Sync web build into native shell
npx cap sync android

# Open in Android Studio
npx cap open android
```

## Key Files (after `npx cap add android`)
```
native/android/
├── app/
│   ├── src/main/
│   │   ├── AndroidManifest.xml    ← Deep links, intent filters, permissions
│   │   ├── java/app/klip4ge/      ← MainActivity + Bridge
│   │   └── res/                   ← Icons, splash screens
│   └── build.gradle               ← Signing, build config
└── build.gradle                   ← Root Gradle config
```

## Share Intent Configuration
Add to `AndroidManifest.xml` inside `<activity>`:
```xml
<!-- Handle incoming shares from other apps -->
<intent-filter>
    <action android:name="android.intent.action.SEND" />
    <category android:name="android.intent.category.DEFAULT" />
    <data android:mimeType="text/plain" />
</intent-filter>
<intent-filter>
    <action android:name="android.intent.action.SEND" />
    <category android:name="android.intent.category.DEFAULT" />
    <data android:mimeType="image/*" />
</intent-filter>

<!-- Deep links: klip4ge://share?url=... -->
<intent-filter android:autoVerify="true">
    <action android:name="android.intent.action.VIEW" />
    <category android:name="android.intent.category.DEFAULT" />
    <category android:name="android.intent.category.BROWSABLE" />
    <data android:scheme="klip4ge" />
</intent-filter>

<!-- HTTPS deep links: https://klip4ge.app/share?url=... -->
<intent-filter android:autoVerify="true">
    <action android:name="android.intent.action.VIEW" />
    <category android:name="android.intent.category.DEFAULT" />
    <category android:name="android.intent.category.BROWSABLE" />
    <data android:scheme="https" android:host="klip4ge.app" />
</intent-filter>
```


## Background Sync (WorkManager)
The `@capacitor/background-fetch` plugin uses Android WorkManager.
Minimum interval: 15 minutes (enforced by Android OS).
Configure in `capacitor.config.ts` under `BackgroundFetch`.

## Build & Release
```bash
# Debug APK
cd native/android && ./gradlew assembleDebug

# Release APK (requires signing config in build.gradle)
cd native/android && ./gradlew assembleRelease

# Or use Capacitor CLI
npx cap build android
```
